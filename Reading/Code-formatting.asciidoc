_General_

- Press kbd:[Ctrl+Alt+L] to reformat.
- To disable formatting on a portion of your tex file, wrap the code with

[latex]
----
% @formatter:off
...
% @formatter:on
----

and enable these commens in the settings by going to menu:Settings[Editor > Code Style > Formatter Control] and checking `Enable formatter markers in comments`.
